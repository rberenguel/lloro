# Actions (Chrome extension)

## [`c`] Open VS Code

```
code .
```

## [`g`] Open Github

## [`o`] Open folder

```
open .
```

## [`f`] Run prettier

```
prettier . --write
```

## [`!p`] Palettes

```
ls palettes/ | awk '{printf "\"palettes/%s\", ", $1}' | sed 's/, $//' | awk '{print "export const BUILT_IN_PALETTES = [" $0 "];"}'
```

## [`!s`] HTTPS Go server

```
cd .. ; go run server.go tessella/
```
